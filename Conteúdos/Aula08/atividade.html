Aprendendo novas tags

table

A tag <table> </table> é utilizada para criar tabelas que representam dados em duas dimensões ou mais.

Sua estrutura básica é composta por duas tags: <tr> que representa uma linha e <td> que representa uma célula. Obs.: A quantidade de colunas é criada automaticamente a partir da quantidade de células.

Exemplo:

<table border="1"> <tr> <td></td> <td></td> <td></td> </tr>
